Hỗ trợ GhiCa

GhiCa hoạt động offline và không yêu cầu tài khoản. Khi cần hỗ trợ, hãy gửi phiên bản ứng dụng, model iPhone, phiên bản iOS và các bước gây ra lỗi.

Không gửi file backup, mật khẩu backup hoặc mã PIN qua email.

Liên hệ hỗ trợ

Backup và khôi phục

GhiCa không thể khôi phục mật khẩu backup bị quên. Hãy kiểm tra file backup trước khi xóa dữ liệu hoặc chuyển sang thiết bị mới.

Xóa dữ liệu

Mở Cài đặt → Quản lý dữ liệu để xóa dữ liệu trong ứng dụng. Gỡ ứng dụng cũng xóa dữ liệu cục bộ; các file backup đã xuất phải được xóa riêng.

Backup and restore

GhiCa cannot recover a forgotten backup password. Verify your backup before erasing data or moving to a new device.

Delete data

Open Settings → Data Management to erase app data. Uninstalling the app also removes local data; exported backup files must be deleted separately.
